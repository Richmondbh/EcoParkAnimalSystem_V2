# EcoPark Animal Management System V1

A Windows Forms application demonstrating Object-Oriented Programming in C# as part of "Programming in C# II" course Assignment requirement.

## Features
- Manage multiple animal species
- Class hierarchy with inheritance
- Polymorphism and dynamic binding
- Interface implementation
- Collection management

## OOP Concepts Demonstrated
-  Encapsulation (private fields, public properties)
-  Inheritance (Animal → Mammal → Cat)
-  Polymorphism (ToString() chain)
-  Interfaces (IAnimal, etc.)
-  Factory Pattern


## Technologies
- C# / .NET 8
- Windows Forms
